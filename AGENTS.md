# Codex 修改規則

請依照以下規則維護本 repo：

## 專案目的

這是給主持人在平板上使用的網頁手卡，重點是現場好讀、好操作、低失誤。

## 主要檔案

- 主要修改檔案：`sankuang-midsummer-host-card.html`
- 首頁入口：`index.html`
- 請不要任意刪除 `.github/workflows/pages.yml`

## 修改原則

1. 優先維持單一 HTML 檔，避免平板離線或網路不穩時出問題。
2. 不要加入外部 CDN、外部字型、外部 JavaScript 套件。
3. 不要使用需要安裝或 build 的框架。
4. 字體要大，平板橫放時需能清楚閱讀。
5. 每個串場必須能獨立切換。
6. 必須保留：
   - 上一段／下一段
   - 段落跳選
   - 字體大小調整
   - 深色／淺色模式
   - 全螢幕
   - 鎖定模式
   - 現場救援短句
7. 舞台提示、男主持、女主持、合講、觀眾互動要清楚區分。
8. 修改主持詞時，不要自行大幅刪改橋段，除非使用者明確要求。
9. Commit 訊息請簡潔描述修改內容。

## 測試重點

修改後請確認：

- `index.html` 可以導向 `sankuang-midsummer-host-card.html`
- 手機和平板直式、橫式都能閱讀
- 按鈕不會遮住主要文字
- 上一段／下一段功能正常
- 救援短句可以開啟和關閉
- 全螢幕失敗時不影響其他功能
